

## Luming 预置的 css 缩写

| 缩写 | 说明   | 对应 CSS 属性      | 后继数值字符串              |
| ---- | ------ | ------------------ | --------------------------- |
| `bg` | 背景色 | `background-color` | css 颜色代码                |
| `rd` | 圆角   | `border-radius`    | 如果没有单位，会自动添加 px |


没有 `bd` 缩写，因为 Luming 在预览模式下会自动为每个组件添加边框以便于区分组件层级关系，如果用户需要自定义边框样式，可以直接使用 CSS 样式修饰语法，例如：`A: border 1px solid #000;`。 // Think again
